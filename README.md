# Haptic Finance

Homepage version 1

## Documentation

### Dependencies

* Node
* React
* Chakra

### Installation

*   Install NodeJs from [NodeJs Official Page](https://nodejs.org/en/)
*   Run `npm install`
*   Run `npm start`
*   Run `npm run install:clean` (if you use a linux based terminal) which will install `node_modules` and also will start your project.
